## 概述

卡证使用文档抽取接口。

```http
POST /open/agentic_doc_processor/{tenant_name}/v1/app/doc/extract
```

异步任务使用相同的请求体：

```http
POST /open/agentic_doc_processor/{tenant_name}/v1/app/doc/extract/create/task
GET /open/agentic_doc_processor/{tenant_name}/v1/app/doc/extract/query/task/{task_id}
```

## 请求参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| `tenant_name` | string | 是 | 租户名称，默认 `laiye`。 |
| `app_id` | string | 是 | 卡证预置应用的唯一标识。 |
| `file_base64` | string | 可选 | 待抽取文件的 Base64，与 `file_url` 二选一。 |
| `file_url` | string | 可选 | 待抽取文件 URL，与 `file_base64` 二选一。仅支持 HTTP 开头的线上地址。 |
| `with_rec_result` | boolean | 否 | 是否返回文档解析结果，默认 `false`。 |

## 支持的卡证类型

| 卡证类型 | 典型字段 |
| --- | --- |
| 车辆合格证 | 合格证编号、车辆制造企业名称、车辆品牌、车辆型号、车辆识别代号/车架号、发动机号、车辆制造日期 |
| 港澳通行证 | 姓名、英文姓名、出生日期、性别、有效期限、签发机关、证件号码 |
| 户口本 | 户别、户主姓名、户号、住址、姓名、与户主关系、身份证号码、登记日期 |
| 驾驶证 | 驾驶证证号、姓名、性别、国籍、出生日期、准驾车型、签发机构、档案编号 |
| 开户许可证 | 核准号、编号、基本存款账户、法定代表人、开户银行、账号 |
| 身份证 | 姓名、性别、民族、出生日期、住址、身份证号码、签发机构、有效期限 |
| 行驶证 | 号牌号码、车辆类型、所有人、使用性质、品牌型号、车辆识别代号、发动机号码、注册日期 |
| 银行卡 | 卡号、发卡机构、有效期、卡类型 |
| 营业执照 | 统一社会信用代码、名称、类型、经营范围、注册资本、法定代表人 |
| 护照-中国 | 姓名、英文姓名、性别、出生日期、出生地点、护照号码、签发地点、有效期 |
| 组织机构代码证 | 组织机构代码、机构名称、机构类型、地址、有效期、颁发单位 |

## 返回结果

返回体使用文档抽取接口的通用结构：

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| `code` | string | 业务码。成功为 `success`。 |
| `message` | string | 返回信息。 |
| `data.task_id` | string | 抽取任务 ID。 |
| `data.status` | integer | 抽取状态。`4` 表示成功，`5` 表示失败。 |
| `data.extraction_result` | array | 卡证字段抽取结果。 |
| `data.classify_result` | object | 文档分类结果。 |
