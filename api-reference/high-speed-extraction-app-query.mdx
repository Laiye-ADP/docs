高速版自定义抽取应用的列表和配置使用两个接口：

| 目标 | OpenAPI |
| --- | --- |
| 查询高速版应用列表 | `GET /open/agentic_doc_processor/{tenant_name}/v1/app-list?ai_function=1425` |
| 查询高速版应用配置 | `POST /open/agentic_doc_processor/{tenant_name}/v1/app-manage/config` |

<Note>
  查询应用列表时，用 `ai_function=1425` 过滤高速版自定义抽取应用。查询应用配置时，只需要传高速版应用的 `app_id`。
</Note>

## 查询高速版应用列表

```http
GET /open/agentic_doc_processor/{tenant_name}/v1/app-list?ai_function=1425
```

### Query 参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| `ai_function` | integer | 是 | 高速版自定义抽取应用固定传 `1425`。 |
| `app_name` | string | 否 | 按应用名称过滤，支持模糊匹配。 |
| `app_type` | integer | 否 | 应用类型。自定义应用传 `1`。 |
| `with_extra` | integer | 否 | 是否返回扩展信息，默认 `1`。 |
| `offset` | integer | 否 | 分页偏移量，默认 `0`。 |
| `limit` | integer | 否 | 每页数量。 |

### 请求示例

```bash
curl --request GET \
  --url 'https://adp.laiye.com/open/agentic_doc_processor/laiye/v1/app-list?ai_function=1425&app_type=1&offset=0&limit=20' \
  --header 'X-API-KEY: {api_key}'
```

### 响应示例

```json
{
  "code": "success",
  "message": "",
  "tips": null,
  "data": {
    "list": [
      {
        "id": "app_1425",
        "app_name": "高速版发票抽取应用",
        "ai_function": 1425,
        "app_label": ["invoice", "high-speed"]
      }
    ],
    "total": 1
  }
}
```

## 查询高速版应用配置

```http
POST /open/agentic_doc_processor/{tenant_name}/v1/app-manage/config
```

### 请求参数

| 名称 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| `tenant_name` | string | 是 | 租户名称，默认 `laiye`。 |
| `app_id` | string | 是 | 高速版自定义抽取应用 ID。 |
| `config_version` | string | 否 | 配置版本。不传时返回最新配置版本。 |

### 请求示例

```bash
curl --request POST \
  --url 'https://adp.laiye.com/open/agentic_doc_processor/laiye/v1/app-manage/config' \
  --header 'Content-Type: application/json' \
  --header 'X-API-KEY: {api_key}' \
  --data '{
    "app_id": "app_1425"
  }'
```

### 响应示例

```json
{
  "code": "success",
  "message": "",
  "tips": null,
  "data": {
    "app_id": "app_1425",
    "app_name": "高速版发票抽取应用",
    "app_label": ["invoice", "high-speed"],
    "config_version": "v1",
    "extract_fields": [
      {
        "field_name": "发票号码",
        "field_type": "string",
        "field_prompt": "抽取发票上的唯一发票号码。"
      }
    ],
    "parse_mode": "standard",
    "enable_long_doc": false,
    "long_doc_config": null
  }
}
```

## 业务规则

1. 查询高速版应用列表使用 `GET /v1/app-list`，并设置 `ai_function=1425`。
2. 查询高速版应用配置使用 `POST /v1/app-manage/config`。
3. 查询配置时不需要传 `ai_function`，服务按 `app_id` 定位应用。
4. 不传 `config_version` 时，接口返回最新配置版本。
5. 高速版应用不记录标准版高级解析配置，配置查询中的 `parse_mode` 可能返回默认值 `standard`。
