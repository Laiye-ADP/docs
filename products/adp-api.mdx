## OpenAPI 使用指南

### 如何获取API Key?

API Key是ADP 用户身份唯一表示

ADP 平台内的应用支持以 API 方式集成到第三方应用。调用 API 前，你需要前往 [ADP 平台](https://adp.laiye.com/) 获取 API Key。点击个人头像后，进入 **我的 API** 查看令牌。

## 接口参考

你可以在左侧导航的 **ADP API** 下查看所有接口。每个接口页面包含请求参数、响应结构和调用示例。

## 认证方式

文档抽取、文档解析、应用管理和积分查询接口使用 `X-API-KEY` 请求头。

工作流接口使用 `X-API-Key` 请求头。

## 基础地址

```text
https://adp.laiye.com
```

## 文件传入方式

文档抽取和文档解析接口支持两种文件传入方式：

- `file_url`：线上文件 URL。仅支持 HTTP 开头的地址。
- `file_base64`：本地文件的 Base64 内容。

`file_url` 和 `file_base64` 二选一即可。
