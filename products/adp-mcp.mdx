## 概述

**adp-mcp** 是 Laiye ADP 的 [Model Context Protocol](https://modelcontextprotocol.io/) 服务器，将 ADP 的文档解析、抽取与人机复核能力以 MCP 工具的形式暴露给 LLM 客户端（Claude Desktop、VS Code、Cursor、Codex CLI 等），让 Agent 可以直接发现、描述、验证与执行 ADP 任务。

<Card title="PyPI" icon="python" href="https://pypi.org/project/adp-mcp/">
  pip install adp-mcp
</Card>

## 核心工具

| 工具 | 说明 |
| --- | --- |
| `adp_discover` | 列出可用 ADP 资源，支持按 domain、intent class、关键字过滤 |
| `adp_describe` | 获取资源 \+ intent class 的使用契约（字段 schema、谓词、投影） |
| `adp_validate` | 验证 intent IR 是否匹配资源 schema，不执行 |
| `adp_execute` | 执行 ADP intent 并返回结果（支持 cursor 分页） |

## 安装

```bash
pip install adp-mcp
```

## 运行

```bash
adp-mcp --config /path/to/adp/config [--log-level INFO]
```

日志默认输出到 stderr，可重定向到文件：

```bash
adp-mcp --config /path/to/adp/config 2>/path/to/adp-mcp.log
```

### 身份验证

使用 Basic Auth，设置环境变量：

```bash
export ADP_USERNAME=myuser
export ADP_PASSWORD=mypassword
```

<Info>
  若未设置 `ADP_USERNAME`，服务器将以匿名方式连接。
</Info>

## MCP 客户端配置

<Tabs>
  <Tab title="Claude Desktop">
    添加到 `claude_desktop_config.json`：

    ```json
    {
      "mcpServers": {
        "adp": {
          "command": "adp-mcp",
          "args": ["--config", "/path/to/adp/config"]
        }
      }
    }
    ```
  </Tab>
  <Tab title="VS Code">
    工作区 `.vscode/mcp.json` 或用户级 `settings.json`：

    ```json
    {
      "servers": {
        "adp": {
          "type": "stdio",
          "command": "adp-mcp",
          "args": ["--config", "/path/to/adp/config"]
        }
      }
    }
    ```
  </Tab>
  <Tab title="Codex CLI">
    `~/.codex/config.toml`：

    ```toml
    [mcp_servers.adp]
    transport = { command = "adp-mcp", args = ["--config", "/path/to/adp/config"] }
    enabled = true
    ```
  </Tab>
  <Tab title="Cursor">
    在 Cursor Settings 中添加 MCP Server：

    ```json
    {
      "mcpServers": {
        "adp": {
          "command": "adp-mcp",
          "args": ["--config", "/path/to/adp/config"]
        }
      }
    }
    ```
  </Tab>
</Tabs>

## Docker 快速启动

仓库 `docker/` 目录提供了一套自包含后端服务与 `mcp-runtime` 容器的 Docker Compose 栈：

```bash
cd adp-mcp/docker
docker compose up -d
```

## 开发

```bash
cd adp-mcp
uv sync --extra dev
uv run python -m unittest discover -s tests -v
```
