---
title: "クイックスタート"
description: "成功への最短ルートを示すガイドから始めましょう"
---

ユーザーが製品を使い始める方法を説明します。最初に何をする必要がありますか。前提条件はありますか。

クイックスタートでは、ゼロから製品を使い始めるまでをカバーします。ユーザーは短時間で成功体験を得て、できることのイメージをつかめます。

## 前提条件

開始する前に、次のものが必要です。

- 要件 1（例: Node.js 18 以降、無料アカウント、API キー）
- 要件 2（例: 対応デバイス、対応ブラウザー、対応 OS）

## はじめに

<Steps>
  <Step title="インストール">
    製品のインストール方法またはサインアップ方法を説明します。

    ```bash
    npm install your-package
    ```
  </Step>
  <Step title="設定">
    初回利用前に必要なセットアップや設定を説明します。

    ```bash
    your-cli init
    ```
  </Step>
  <Step title="実行">
    ユーザーが動作を確認するために最初に行う操作を示します。

    ```bash
    your-cli start
    ```
  </Step>
</Steps>

<Tip>
  ユーザーがヘルプを得られる手段を用意しましょう。サポートページへのリンク、サポート担当者とのチャット、製品フォーラムなどが考えられます。

  例: お困りですか。[support@yourcompany.com](mailto:support@yourcompany.com) までご連絡ください。
</Tip>
